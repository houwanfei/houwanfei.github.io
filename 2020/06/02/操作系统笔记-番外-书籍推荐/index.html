操作系统笔记-番外-操作系统经典书籍推荐

Posted on 2020-06-02 | In OS学习笔记

最近整理以前的笔记，有人问关于操作系统的书籍。我有个爱好喜欢收集书籍，前后也收集了几百本高质量的书籍，这里给大家推荐基本关于操作系统的书籍

Operating System Concepts 10th edition

又称恐龙书，这本书已经出到第10版，可见其经典。作者是想从理论层面把问题的产生和解决思路阐述清楚，包含了操作系统各个方面，是一本非常不错的入门书籍。

豆瓣书评

下载地址：

https://github.com/woodliu/Operating.System.Concepts.10th.Edition/blob/master/Operating.System.Concepts.10th.Edition.2018.4.pdf

Modern Operating Systems 4th Edition

光看作者的名字Tanenbaum就知道这本书有点东西，MINIX最初的作者。写作语言通俗易懂，即使你英语不是很好。Processes and Threads, Memory Management, File Systems, Input/Output,DeadLocks这几章写得非常不错，建议多读几遍。

豆瓣书评

下载地址：

链接：https://pan.baidu.com/s/1UCyriXIWRC2-HXsd2DgaYw

提取码：7m8j

Operating System : Three easy pieces

这本书就是最好的入门教程, 所有细节讲的非常清楚, 非常的深入浅出, 强烈推荐操作系统入门玩家阅读。主要分为三部分，虚拟化、并发和持久化来讲。我当初最先读的是《Modern Operating Systems》，如果是我现在推荐你读，肯定是推荐你先看这本。

豆瓣书评

下载地址：

链接: https://pan.baidu.com/s/1YiB9WWGhJBmgmQgrNWPgXg

提取码: 477f

Computer Systems: A Programmer’s Perspective 3rd Edition

以程序员的视角理解计算机系统，首先把高级语言转换成计算机能够理解的一种中间格式（如汇编）， 然后描述计算机如何解释和执行这些中间格式的程序，使系统的那一部分影响程序的执行效率。 总的来说，《深入理解计算机系统》艺术是一个桥梁，帮助程序员衔接了计算机系统各个领域的知识， 为程序员构造了一个概念性的框架。

豆瓣书评

下载地址：

https://545c.com/file/15081096-306053388
